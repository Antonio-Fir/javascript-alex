Operadores de aritmeticos

Además de las operaciones aritmeticá estandar (+, -, *, /). Javascript brinda los siguientes operadores aritmeticos:

Operadores de concatenación

Une dos valores de tipo string, devolviendo otro string correspondiente a la unión.

Ejemplo:
console.log("mi" + "string"); // mi string

Operadores de ternario

Necesita 3 operandos:
condición? valor 1: valor 2
Si la condición es true tomara el valor 1, sino el valor 2.
